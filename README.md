# 1.工程简介说明

本工程主要为自己学习和研究工程，工程管线为unity默认管线

## 1.1 工程结构说明

 工程内MyAsset为自己编写资源  
 工程内命名方式，为 目的_内容_编号  
 工程内目的编号含义 L为学习 R为记录  
 工程内内容编号含义 S为shader G为git资源  
 工程内编号含义 排序内容
